# Soul Sphere
[WebGL](https://en.wikipedia.org/wiki/WebGL) user visualisation interface based on [three.js](https://github.com/mrdoob/three.js/) library

### Run SoulSphere App

#### testing environment with hot reloading <br>
- install dependencies
``% yarn install``

- run server
``% yarn dev``
